# full-wave-rectified-dc-power-supply
Designed a full-wave rectifier DC power supply using 1N4148 diodes, a 25 uF RC filter, and a 300 ohm load to convert a 1 kHz AC input into a stable DC output. This was done for ELECENG 2EI4: Electronic Devices and Circuits I
